STACK & ARCHITECTURE
NestJS 11 + Prisma 7 + PostgreSQL 17 + Docker, modular monolith, npm workspaces (apps/api=@hrms/api). Schema ground-truth at apps/api/prisma/schema.prisma.

Prisma 7 (Rust-free/ESM): client via PRISMA token as ExtendedPrismaClient; typed data/where use as never, results as unknown as T.
CryptoService @Global: encrypt/decrypt async, blindIndex/isEncrypted sync. Encrypted employee PII: nationalId/kraPin/bankAccountNumber. Org's own kraPin/bankAccountNumber PLAINTEXT.
Auth: JWT {sub,org,role} 15min + opaque refresh; scrypt. HR_MANAGEMENT_ROLES=['Admin','HR Manager','HR Officer']; AuthUser={userId,organizationId,role}.
Tenant scoping (src/prisma/tenant-scope.ts, TENANT_SCOPED_MODELS): Payslip/BankExportBatch scoped VIA PayrollRun (extension does NOT inject org into nested relation filters — to bound a Payslip aggregate, resolve org-scoped PayrollRun IDs first, then query payslips by IDs).
Immutability triggers on payroll_runs/payslips/audit_logs.
